TDJANGOH Usage Instructions

Requirements

GCC > 4.4.7 with build-in GFORTRAN

ROOT > 5.34/32

LHAPDF 5.9.1 (in any case lower than LHAPDF 6.x.x) Download LHAPDF 5.x.x

CERNLIB (2006) Download CERNLIB

(Tested on LXPLUS and CCAGE)

Environment variables

After recovering the package, export the following variables (better in .bashrc or similar) :

export TDJANGOH=/path/to/TDJANGOH/ export LHAPDF5=/path/to/LHAPDF5/

Setup & Build

To setup : make setup

To build : make
